# uptime plugin for Mackerel
## Example

```
 $ mackerel-plugin-uptime
uptime.uptime_sec	84380	1506835431
 $ cat /proc/uptime
84379.58 69287.34
 $ uptime
 14:23:51 up 23:26,  2 users,  load average: 0.79, 1.11, 1.21
```
